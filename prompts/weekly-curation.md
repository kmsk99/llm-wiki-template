# Weekly Curation Prompt

이 repo의 최근 변경을 주간 점검해줘.

해야 할 일:
1. 최근 추가되거나 수정된 raw / wiki 문서를 훑는다.
2. 중복되거나 충돌하는 설명이 있는지 찾는다.
3. 빠진 decision, playbook, glossary, entity 문서를 제안하거나 직접 추가한다.
4. 끊어진 링크나 비어 있는 source/frontmatter를 보완한다.
5. `index/` 문서가 최신 상태인지 확인하고 업데이트한다.
6. 불확실한 내용은 확정하지 말고 open question으로 남긴다.

최종 출력 형식:
- 이번 주 정리 요약
- 생성/수정 파일 목록
- 병합 또는 분리한 문서
- 남은 리스크 / open question
